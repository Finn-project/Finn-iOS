### 사용된 개념들

1. Cocoapods을 이용한 오픈소스 라이브러리 사용 <br>
2. Alamofire 이용한 네트워크 통신 <br>
3. FacebookLogin SDK를 이용한 로그인 방식 적용 <br>
4. MapKit 이용
	-	Camera, Annotation, Geocode <br>
5.	회원가입 에러처리 Animation 적용
6.	FSCalender 적용
